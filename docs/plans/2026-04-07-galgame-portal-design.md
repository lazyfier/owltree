# Galgame Portal Visual Design

## Goal
Transform the current Owltree portal into a polished galgame / visual novel menu-inspired interface without adding new modules, routes, or content blocks.

## Direction
This redesign should feel like the main menu and route selection screens of a refined AVG / galgame rather than a generic SaaS dashboard or a dark AI template. The design should be expressive and recognizably 二次元, but still clean enough for notes, tools, trends, and game pages to remain usable.

## Tone
- Soft, romantic, slightly dramatic
- More menu-screen elegance than gamer HUD clutter
- A personal otaku atmosphere without character art overload

## Visual Language

### 1. Surfaces
- Cards should feel closer to dialogue windows, save/load panels, or route selection boxes.
- Use layered borders, soft inner glow, light translucency, and cleaner framing.
- Corners can become a little sharper or more ornate, but not overly decorative.

### 2. Color
- Shift from misty teal-cyber into a galgame-friendly palette:
  - cream white
  - sakura pink
  - lilac / soft violet
  - silver-gray
  - keep a little cyan only as a technical highlight
- Pink/lilac should become part of the atmosphere, not a loud candy gradient everywhere.

### 3. Typography Hierarchy
- Main headings should feel like route titles / chapter headers.
- Subtitles should feel like menu labels or episode copy.
- Labels, pills, and metadata should look like route tags or system markers.

### 4. Motion
- Hover states should feel like menu selection states.
- Use subtle glow, border brightening, and vertical lift.
- Avoid noisy animations or cluttered effects.

## Scope
- Homepage hero, category cards, and trends preview
- Workspace pages: Games, Notes, Tools, Trends
- Shared visual styling through existing global styles where useful

## What Must Not Change
- No new modules or sections
- No new routes
- No new feature logic
- No character illustrations or extra fandom widgets
- No loss of readability or usability

## Success Criteria
- The site feels recognizably more personal and anime-coded through style alone.
- The cards feel like galgame UI panels, not generic dashboard cards.
- The portal still reads as a usable multi-page personal site, not a parody theme.
